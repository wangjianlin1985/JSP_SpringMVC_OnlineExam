# JSP_SpringMVC_OnlineExam
JSP基于SpringMVC在线考试系统设计毕业源码案例设计

## 功能说明：
  学生角色包含以下功能：学生参加考试界面,查看考试记录等功能。
  教师角色包含以下功能：修改密码,查看试卷列表,查看题库,编辑试卷列表,登录页面等功能。
  管理员角色包含以下功能：专业管理,修改密码,学生管理,年级管理,教师管理,班级管理等功能。
